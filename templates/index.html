About the Project

AI Missing Person Detection System is a smart web application developed to simplify the process of registering and searching missing persons. The system stores the details securely in a SQLite database and allows users to upload photographs for future identification. The project is developed using Python, Flask, SQLite and OpenCV. It provides a user-friendly interface for registration, searching and future webcam-based face recognition. This application aims to reduce manual effort and improve the efficiency of missing person management using Artificial Intelligence technologies.
