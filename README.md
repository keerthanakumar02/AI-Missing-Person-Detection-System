# 🛡 AI Missing Person Detection System

## 📌 Project Overview

AI Missing Person Detection System is a web application developed using Python and Flask to help register and manage missing person records digitally.

The application allows users to register missing persons, upload photographs, search records, and manage data efficiently using SQLite.

---

## 🚀 Features

- 📝 Register Missing Person
- 📷 Upload Photo
- 🔍 Search Missing Person
- 🗑 Delete Record
- 📊 Dashboard Statistics
- 💾 SQLite Database Storage
- 🎨 Responsive User Interface

---

## ▶ How to Run

1. Clone the repository

```
git clone https://github.com/keerthanakumar02/AI-Missing-Person-Detection-System.git
```

2. Open project folder

```
cd AI-Missing-Person-Detection-System
```

3. Install dependencies

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
